## 2Ô∏è‚É£ Why use Express.js?

* Reduces boilerplate code

* Widely used in industry

## 3Ô∏è‚É£ Express vs Node.js

* **Node.js** ‚Üí runtime environment

## 6Ô∏è‚É£ Routing

Routing means handling different URLs.

```js
app.get("/", (req, res) => {
  res.send("Home Page");
});

app.post("/login", (req, res) => {
  res.send("Login Page");
});
```

### HTTP Methods

* `GET` ‚Üí fetch data
* `POST` ‚Üí send data
* `PUT` ‚Üí update data
* `DELETE` ‚Üí remove data

---

## 7Ô∏è‚É£ Request & Response

* `req` ‚Üí request from client
* `res` ‚Üí response from server

```js

req.query
req.body
```

---

## 8Ô∏è‚É£ Middleware

Middleware runs **between request and response**.

```js

```

Types:

* Built-in
* Custom
* Third-party

---

## 9Ô∏è‚É£ Sending Responses
