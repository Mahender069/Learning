## What are **application-level settings** in Express?

### ðŸ‘‰ Simple definition

> **Application-level settings are configurations applied to the entire Express app using the `app` object.**

They affect **how the whole application behaves**, not just one route.

---

## Where are they defined?

Usually in the **app module** (`app.js`), right after creating the app:

```js
const express = require("express");
const app = express();
```

---

## How do we set application-level settings?

Using:

```js
app.set(settingName, value);
```

And read using:

```js
app.get(settingName);
```

---

## Common application-level settings

### 1ï¸âƒ£ Port

```js

```

---

### 2ï¸âƒ£ View engine

```js

```

---

### 3ï¸âƒ£ Views folder

```js

```

---

### 4ï¸âƒ£ Environment

```js

```

Check it:

```js

```

---

### 5ï¸âƒ£ Case-sensitive routing

### 6ï¸âƒ£ Strict routing
